# H BRIDGE
**What is an H Bridge?**

- H Bridge is a type of circuit that can be used to control speed and rotation of a DC motor, the speed is controlled with PWM (Pulse With Modulation), therefore it can be connected to a microcontroller like Arduino or ESP, if not the direction can be controlled for example by a button. (Essentially, you use the H-bridge to choose the direction (forward/reverse) and the PWM signal to set the speed).
What use does an H Bridge have?

 - Speed control
 - Automation
 - Robotics

**What components does an H bridge consists of?**

- Usually four mosfets (or BJTs in low power and older design, or even IGBTs for high power or industrial applications)
- Flyback diodes (motor acts as an inductive load, that can generate voltage spikes, that can kill the mosfets)
- Capacitors to filter the noise generated by the motor
- Resistors if BJTs are used to limit the base current
